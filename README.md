# send-sms-message
Send a SMS message from AWS Lambda with python and Twilio.

## Requirements
- Python 3.9
- Pip3

## Instalation 
```
pip3 install -r requirements.txt --target .
````
ou...

``` 
pip3 install --target . python-dotenv
``` 
``` 
pip3 install --target . twilio
``` 
## Run local
``` 
python3 index.py
``` 

## Config AWS Lambda
Compress all files and upload to lambda (.zip)
